# 迭代與遞迴
## 迭代

####  迭代是一種利用for或while迴圈執行“重複”的動作以下用費氏數列舉例：
###### 題目：計算一個非負整數並印出第 n 個斐波那契數:

```python
target = int(input()) #第n個費氏數列
n_1 = 0 #定義費氏數列第０項
n_2 = #定義費氏數列第１項
counts = 0 #計算累積次數
if target == 0: #初始測資為０的例外條件
    print(0)
else:
    while counts < target - 1: #在累積次數尚未達成目標時進行“重複”算
        counts +=1 #每次計算時次數＋１，以便能夠判斷終止時機
        n_1,n_2 = n_2,n_1+n_2 #將每次迭代時的指標位置執行的前兩項相加計算下一項，並將後項與前項交換，將後項儲存為下一項
    print(n_2) #列印出目標位置之值
```
# 遞迴
#### 遞迴是將大問題猜解成小問題，並讓程式更加簡潔並且結構化
##### 遞迴能夠增加程式的可讀性
##### 遞迴不用使用for迴圈和while迴圈，是在函式內自我呼叫
###### 題目：計算一個非負整數並印出第 n 個斐波那契數:
```python
n = int(input()) #第n個費氏數列
def fib(n): #定義費氏數列以便進行
    if n == 0: #設定終止條件避免陷入無窮遞迴
        return 0
    if n == 1: 
        return 1
    return fib(n-1) + fib(n-2) 
print(fib(n))#輸出答案
```
